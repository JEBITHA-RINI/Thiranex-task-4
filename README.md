# **Haze & Hue** 🌤️

**Haze & Hue** is a **  real-time Weather Dashboard** designed to provide accurate meteorological data through a visually analysis, user-centric interface.

## 🚀 **Key Features**

* **Live Weather Integration**: Fetches precise, real-time data including **Temperature, Humidity, Wind Speed, Sunrise, and Sunset** using the OpenWeatherMap REST API.
* **Dynamic Theme Engine**: Offers a seamless **Dark/Light Mode toggle** with a "Modern Light Blue" aesthetic for daytime and a "Professional Dark" theme for reduced eye strain.
* **Advanced Glassmorphism UI**: Built with a **modern, blurred glass effect** that creates a premium, high-end look suitable for any developer portfolio.
* **Persistent Search History**: Utilizes **Browser LocalStorage** to save and display your recent city searches, allowing for **quick-click retrieval** of past weather data.
* **Smooth Motion Design**: Features an **animated, moving gradient background** that elevates the dashboard beyond static web pages.

## 🛠️ **Technical Specifications**

* **Core Technologies**: **HTML5, Modern CSS3, and Vanilla JavaScript (ES6+)**.
* **Asynchronous Logic**: Implemented using the **Fetch API and async/await** patterns for non-blocking network requests.
* **Data Handling**: Efficient parsing of **nested JSON objects** to extract specific weather metrics.
* **Robust Error Handling**: Includes **try/catch blocks** to gracefully manage failed network requests and invalid user inputs.

Built with care for simplicity and performance.
